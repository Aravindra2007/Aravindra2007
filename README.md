## 🌟 Featured Project

### 🐆 JAGUAR_AI

*My main, actively-evolving project*

A personalized desktop AI assistant built in Python, designed to run natively on my own system — voice interaction, automation, memory, and more, all wired together into one app.

**What it's built with:**

🎙️ Speech Recognition + Text-to-Speech &nbsp;|&nbsp; 🤖 OpenAI / Ollama backends &nbsp;|&nbsp; 🖥️ Desktop Automation
👁️ OCR & Vision (EasyOCR, OpenCV) &nbsp;|&nbsp; 📄 PDF Reading &nbsp;|&nbsp; 🔔 Wake-Word Detection &nbsp;|&nbsp; 🧠 Persistent Memory

[📂 View Code](https://github.com/Aravindra2007/JAGUAR_AI)

> This project is under active development — expect frequent updates, refactors, and new modules as it grows. It's effectively the successor to my earlier `Voice_Assistant` project below, rebuilt with a more modular architecture (`core`, `agents`, `plugins`, `skills`, `memory`, `scheduler`).
